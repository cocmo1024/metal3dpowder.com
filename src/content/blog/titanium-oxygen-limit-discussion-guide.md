Titanium buyers often ask for "low oxygen" as if that were a complete specification. It is not. Oxygen control matters, but the correct commercial conversation is not about chasing the lowest number in isolation. It is about matching the oxygen expectation to the alloy grade, the project stage, and the release logic the program can actually support.

That is why oxygen discussion should happen early in the RFQ rather than after the supplier has already quoted a generic titanium lot.

## Why oxygen gets so much attention in titanium

Titanium programs pay close attention to oxygen because interstitial control is part of how the alloy route is understood commercially and technically. That does not mean every titanium project should demand the same oxygen expectation. Different grades and different program stages justify different levels of control.

The useful question is not "what is the lowest oxygen number available?" The useful question is "what oxygen and chemistry window does this project actually require?"

## Grade choice changes the conversation

A Ti64 project aligned with a Grade 23 route is not the same sourcing conversation as a broader Grade 5 or commercial-purity titanium inquiry. The existing [Ti64 Grade 23 vs Grade 5 guide](/posts/Alloys/ti64-grade-23-vs-grade-5-powder/) is the right place to frame that difference. The [main TC4 / Ti-6Al-4V guide](/posts/Alloys/tc4-ti6al4v-powder/) also explains why buyers should not ask only for "TC4 powder" without clarifying the chemistry intent.

## Why one universal oxygen limit is the wrong RFQ habit

Buyers sometimes carry an oxygen number from one project into every later inquiry. That is risky. A useful oxygen discussion should consider:

- alloy grade
- intended AM process
- project stage
- whether the route is prototype, qualification, or recurring supply
- what the internal release basis actually requires

If those items are not explicit, the number on the RFQ may look precise while still being commercially unhelpful.

## Oxygen control also interacts with reuse and receiving

Oxygen is not only a quoting issue. It affects how the incoming lot is reviewed and how reused powder is discussed internally. That is why the oxygen question should usually be read alongside the [COA guide](/posts/Alloys/metal-powder-coa-certificate-guide/), the [incoming inspection checklist](/posts/Alloys/incoming-inspection-checklist-am-powder-lots/), and the [powder reuse policy guide](/posts/Alloys/metal-powder-reuse-policy-guide/).

## What buyers should actually tell the supplier

If oxygen matters to the project, the buyer should define:

1. the titanium grade route
2. the process context
3. whether the project is prototype or controlled release
4. whether an internal oxygen ceiling already exists
5. what certificate evidence is required on the lot

That gives the supplier something actionable instead of a vague request for "better titanium."

## The practical takeaway

Titanium oxygen control should be discussed as part of the grade and release strategy, not as an isolated number chase. Buyers get cleaner quotes and more useful lots when the oxygen expectation is tied to the actual alloy route, the actual program stage, and the actual certificate basis.
