# Metal 3D Powder SEO Roadmap

## Working rule

Before publishing a new article:

1. Read the existing titles in `src/content/blog/`.
2. Check whether the search intent already exists as:
   - a material profile,
   - a comparison page,
   - a buying guide,
   - a pricing page,
   - or a quality/process guide.
3. If the intent already exists, update that URL instead of creating a near-duplicate page.
4. Use site pricing from `src/data/products.ts` when mentioning on-site indicative prices.
5. Avoid precise third-party price claims unless they can be verified from live public supplier pages.
