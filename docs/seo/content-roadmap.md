# Metal 3D Powder SEO Roadmap

## Working rule

Before publishing a new article:

1. Read the existing titles in `src/content/blog/`.
2. Keep the topic inside one of these buckets only:
   - material profile,
   - material comparison,
   - process guide,
   - application guide.
3. If the topic drifts into supplier management, pricing, logistics, RFQ administration, or document-control operations, do not publish it.
4. If the search intent already exists, upgrade the existing URL instead of creating a near-duplicate page.

## Current content direction

The blog is now application-first. It should answer:

- what the alloy is,
- where it is used,
- why it fits that application,
- how the AM route changes the result.

It should not drift into supplier-side operations unless the topic is directly about the material or the application itself.
