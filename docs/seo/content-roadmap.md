# Metal 3D Powder SEO Roadmap

## Working rule

Before publishing a new article:

1. Read the existing titles in `src/content/blog/`.
2. Keep the topic inside one of these buckets only:
   - material profile,
   - material comparison,
   - process guide,
   - application guide.
3. If the topic drifts into supplier management, pricing, logistics, RFQ administration, or document-control operations, do not publish it.
4. If the search intent already exists, upgrade the existing URL instead of creating a near-duplicate page.

## Current content direction

The blog is now application-first. It should answer:

- what the alloy is,
- where it is used,
- why it fits that application,
- how the AM route changes the result.

It should not drift into supplier-side operations unless the topic is directly about the material or the application itself.

## Powder-intent principle

The traffic target is still people looking for powder.

That means:

- keep `powder` in titles where the search intent is alloy- or application-led,
- keep the alloy family visible in the first paragraph,
- keep process words like `LPBF`, `EBM`, `DED`, `particle size`, and `oxygen` where they genuinely matter,
- route application pages back into material pages so the search path stays powder-centered instead of drifting into generic hardware content.

## Content audit status

The current stock set is now kept on site because it still fits powder-intent traffic:

- alloy profile pages capture direct powder searches,
- comparison pages capture material-decision searches,
- process pages capture route-specific powder searches,
- application pages capture "alloy + use case" searches without turning into generic procurement content.

The cleanup rule is simple:

- keep technical terms such as `commercially pure titanium` when they are part of the alloy definition,
- remove generic business language such as `buyers`, `commercial offer`, `catalog`, or `supplier-side` wording when it does not help a powder searcher understand the material or the application.
