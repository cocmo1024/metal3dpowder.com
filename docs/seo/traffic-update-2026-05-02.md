# High-Value Traffic Update - 2026-05-02

## Decision

This round adds four material-family pillar pages rather than more narrow application pages.

Reason:

- Search Console already shows early impressions for broad metal powder searches.
- The site has many alloy and application pages, but several material-family query patterns still lacked a dedicated blog landing page.
- These pages can catch broad material-intent searches and then route visitors into specific alloy and application pages.

## Internal-link intent

Each new article links to existing alloy and application pages rather than competing with them. The goal is to create stronger topical paths:

- broad metal powder guide,
- material-family guide,
- alloy profile,
- application page.
